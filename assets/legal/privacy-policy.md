# Privacy Policy – GTM Debug Runner

Last updated: 2025-12-30

## English

### Privacy Policy – GTM Debug Runner

This extension does not collect, store, or transmit any personal data.

All functionality is executed exclusively within the user’s browser. No tracking, user behavior analysis, logging, or communication with external servers takes place.

Only technical settings and runtime state (such as debug progress, loaded URL lists, or UI positions) are stored locally using the Chrome Storage API. This data remains entirely on the user’s device.

No data is shared with third parties.

If you have any questions regarding privacy, you may contact the developer via the contact information provided in the Chrome Web Store listing.

## Deutsch

### Datenschutzerklärung – GTM Debug Runner

Diese Erweiterung erhebt, speichert oder überträgt keine personenbezogenen Daten.

Alle Funktionen der Erweiterung werden ausschließlich lokal im Browser des Nutzers ausgeführt. Es findet kein Tracking, keine Analyse des Nutzerverhaltens und keine Kommunikation mit externen Servern statt.

Gespeichert werden lediglich technische Einstellungen und Laufzustände (z. B. Fortschritt eines Debug-Durchlaufs, geladene URL-Listen oder UI-Positionen), die ausschließlich lokal über die Chrome-Storage-API abgelegt werden.

Es werden keine Daten an Dritte weitergegeben.

Bei Fragen zum Datenschutz kann der Entwickler über die im Chrome Web Store angegebene Kontaktadresse erreicht werden.
